## 使用建议

- 对个人学习者：按“总览 -> 学习路径 -> Labs”走最快
- 对项目经理或 Enablement 负责人：先看“培训体系总览”和“课程安排”
- 对讲师：先看“讲师手册”，再按 cohort 目标挑 Labs 和 Capstone
