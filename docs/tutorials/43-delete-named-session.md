# Lesson 43: 删除一个已经不用的命名会话

教程做到这里，  
命名会话的完整生命周期已经差不多齐了：

- 保存
- 查看
- 导出
- 复用

最后一步就是清理。

目标：

- 删除一个已经不用的命名会话
- 理解删除的到底是“别名”还是“浏览器配置”

## 开始前

建议先完成：

- [Lesson 40](40-save-named-session.md)
- [Lesson 42](42-use-named-session.md)

## Step 1: 删除命名会话

```bash
./tsplay -action delete-session -session-name session_lab_demo
```

预期结果：

- 返回删除结果
- 相关元数据会被清理

## Step 2: 再检查一次

```bash
./tsplay -action list-sessions
```

如果删除成功，那么 `session_lab_demo` 就不应该再出现在列表里。

## Step 3: 这节要带走什么

命名会话不是神秘对象，  
它也是一份可以创建、查看、复用、删除的明确资产。

走完这一步，浏览器状态这条线就已经形成了完整闭环：

1. 观察状态
2. 保存状态
3. 加载状态
4. 注册命名会话
5. 复用命名会话
6. 清理命名会话
