# Lesson 148 First-Run Entry Checklist

新用户第一次运行 `tsplay` 时，优先给他这条顺序：

1. `./tsplay -action list-assets`
2. `./tsplay -action extract-assets -extract-root ./tsplay-assets`
3. `./tsplay -flow script/tutorials/01_hello_world.flow.yaml`
4. 打开 `docs/tutorials/README.md`

这条顺序的目标是：

1. 先看见资源
2. 再把资源释放出来
3. 再建立第一次成功运行
4. 最后再进入完整课程导航
