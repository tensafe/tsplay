# Lesson 137 Add Demo Page Checklist

只有在下面这些情况里，才建议新开一个 demo 页面：

1. 你要教学的是页面状态、选择器、等待、上传、下载、session、storage state 这类浏览器行为。
2. 现有 demo 页面不能稳定复现你要教的现象。
3. 你不是为了“好看”，而是为了复现一个清晰、稳定、可讲解的动作。

下面这些情况，通常不需要新 demo 页面：

1. 只是本地数据读写、文件转换、Redis、DB、artifact 组织。
2. 只是想换一个页面主题，但教学动作没有变化。
3. 现有 demo 已经能复用，只是文档还没写清楚。
